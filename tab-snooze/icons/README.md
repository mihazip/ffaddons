# Icon Assets

## Generating Icons

To generate the required PNG icon files:

1. Open `generate-icons.html` in your web browser
2. The page will automatically generate all icon sizes
3. Click the "Download" button for each icon:
   - moon-16.png
   - moon-32.png
   - moon-48.png
   - moon-128.png
4. Save each file in this directory

## Using SVG (Alternative)

For Firefox development, you can temporarily use the SVG file by updating the manifest.json to reference `moon.svg` instead of PNG files. However, for Chrome compatibility and final distribution, PNG files are required.

## Manual Creation

If you prefer to create your own icon:
- Use any image editor (GIMP, Photoshop, Figma, etc.)
- Create a moon/crescent icon
- Export in the following sizes: 16x16, 32x32, 48x48, 128x128
- Save as PNG files with the names listed above
